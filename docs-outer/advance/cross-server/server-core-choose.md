### 关于 waterfall

[关于 Waterfall 停止维护的公告](https://forums.papermc.io/threads/announcing-the-end-of-life-of-waterfall.1088/)。

:::info

停止维护是指不再为其添加新功能或错误修补，

Waterfall 仍然会合并来自 BungeeCord 的更改。

对于仍然想使用 Waterfall 的人来说这可能不是一个很大的影响。

:::
