# 笨蛋开服教程

👋 欢迎来到笨蛋开服教程！

笨蛋开服教程是一篇主要针对 **高版本 Java 版** 和 **基岩版**服务器的开服指南，而非**远古核心**。

在开始前，请确认您拥有以下基础：

- 具备**基础**的计算机操作技能 - 如解压、新建文件夹、打开文件、安装软件等；
- 具备使用**搜索引擎**搜索关键词的能力；
- 拥有一台能够**正常游玩 Minecraft** 的电脑。
- 具有一定的**英语基础** 

# 教程适合的群体

- 渴望学习如何开设服务器并运营良好的人；
- 寻求解决如何解决服务器的常见问题的人；
- 希望深化对服务器基础知识理解的人。

# 教程不适合的群体

- 意图通过开设服务器进行欺诈谋利的人；
- 忽视插件文档，频繁依赖他人解答的人；
- 头脑简单、狂妄自大、懒惰无能的人。
