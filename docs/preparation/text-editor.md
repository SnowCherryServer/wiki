# 文本编辑器

一个好用的文本编辑器是**相当**重要的，请不要再使用 Windows 自带的记事本和写字板了。~~(如果你想用也不是不行)~~

# 记得保存文件

发现有人第一次用文本编辑器不知道编辑完后保存。

![](_images/白点.png)

VSCode 会在未保存的文件后标记一个白点，提示你这个文件编辑了但是没有保存（其他文本编辑器类似）。

![](_images/vscode保存和自动保存.png)

保存按钮（红色箭头），也可以用快捷键来保存 Ctrl + S 。

开启自动保存（绿色箭头）。
