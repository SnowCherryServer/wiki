目前主流的全息图插件，简称 dh 。

纯发包，不占服务器资源。

可以自定义全息图动效，还能展示各种实体和物品。

支持交互，甚至可以用它来做菜单。

详细请看 [wiki](https://wiki.decentholograms.eu/)，比较好上手

内核汉化请看上方 MineBBS 帖子
